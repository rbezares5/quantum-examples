Number Partitioning Problem¶

Introduction¶

This problem consists on dividing a given set \({S = (a_1 , ... , a_N)}\) of integer numbers into 2 subsets S1 and S2 so that the sum of the elements of both subsets have the same total value. Depending on the initial data this may not always be possible, but it can be optimized so that the difference between the sums of both subsets is minimized.

Formally, the problem can be formulated with the following decision variables:

\[\begin{split}x_i= \begin{cases} -1 & \text{if }x_i \text{ belongs to }S_1 \\ 1 & \text{if }x_i \text{ belongs to }S_2 \end{cases}\end{split}\]

And in this case the objective function to be minimized is:

\[f = (a_1*x_1 + a_2*x_2 + ... + a_N*x_N)^2\]

This is implemented by the optimization_problems.number_partition.number_partition() function.

It can be simply called as it is:

>>> number_partition(data="dummy", sampler="sim")
